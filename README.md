# 📚 Academic Publishing Platform (Soroban Smart Contract)

## 🧠 Project Description

The Academic Publishing Platform is a decentralized application built on the Stellar Soroban smart contract platform. It enables researchers and authors to publish academic papers in a transparent, tamper-proof, and censorship-resistant manner.

This project aims to eliminate centralized control in academic publishing and provide open access to research content using blockchain technology.

---

## ⚙️ What It Does

- Allows users to submit academic papers to the blockchain
- Stores metadata of papers including:
  - Title
  - Author
  - Content reference (IPFS hash or external storage link)
- Enables retrieval of all published papers
- Allows access to individual papers by index

---

## ✨ Features

- 📄 Decentralized paper submission
- 🔐 Tamper-proof storage using blockchain
- 🌐 Integration-ready with IPFS or off-chain storage
- 📚 Public access to all published research
- ⚡ Built on Stellar Soroban (fast and low-cost)

---

## 📦 How to Use

1. Deploy the smart contract using Soroban CLI
2. Call `submit_paper` to publish a paper
3. Use `get_papers` to fetch all submissions
4. Use `get_paper(index)` to fetch a specific paper
